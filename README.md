# Adopt-a-Pet React Application

This repository contains a React application for browsing and searching pets available for adoption. It utilizes `react-router-dom` for client-side routing.

## Features

- **Homepage**: Displays pets categorized by type.
- **Dynamic Paths**: Routes dynamically based on pet types and IDs.
- **Search Page**: Allows users to search for specific pets.
- **Error Handling**: Displays a "Pet Details Not Found" page for invalid pet details.

## Dependencies

- [react](https://reactjs.org/)
- [react-dom](https://reactjs.org/docs/react-dom.html)
- [react-router-dom](https://reactrouter.com/web/guides/quick-start)
